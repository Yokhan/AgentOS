# Technical Writing Review

Read the shared writing SOT, technical profile, reference registry, editorial
board, and product evidence. Report findings first for four separate lanes:
technical accuracy, procedure execution, information architecture, and technical
language. State what was executed versus inspected. A writer self-check,
generated reference, or unrun command is not independent acceptance.
